# 第2章 祠堂议并

祖祠里那句话落下后，雨声像一下子压到了每个人头顶。

陆承前坐在长案边，掌心按着冰凉案角，指节却慢慢攥紧了。

他还没完全适应这副身体。

胸口时不时发闷，后背一阵阵发虚，脑子里那点属于原身的记忆也依旧是碎的。可他至少已经分清楚了眼下最要命的事。

今晚要真让人收了祖祠，这一脉就算还有人活着，往后也多半只是挂在主支账册下的一群闲人。

陆伯衡没急着开口，只用那双浑浊却还算稳的眼睛看了看门口的主支来人，又转头看向堂内众人。

陆正松先一步接了话。

“主支既是为我等着想，倒也不是坏事。”他捋了捋袖口，声音不高，却故意说得人人都能听清，“如今族里什么样，大家也都看见了。药泉快断，库里见底，田上连人都凑不齐。再撑下去，怕是连命都没得保。并脉虽难听些，总好过整个旁支一起烂死。”

他这番话显然不是临时起意。

祖祠里好几个人都跟着低下头，有人眼里犹豫，有人则像早就想好了，只差一个先开口的人。

“正松叔这话也没错……”

“主支既愿收我们，至少孩子们还能有条修路。”

“守着这祠，守得住么？”

一时间，祖祠里说什么的都有。

陆承前听着这些声音，脑子里那股乱劲先退了些。

前世他刚接保安队那阵子，一出事也是这样。人一多，嘴就杂，有人喊干不动，有人说算了，真让这口气散了，后面什么都收不住。

更要命的是，陆正松说的并不全是假话。

原身留下来的那点记忆碎片里，旁支眼下是真快见底了。

祖地灵脉本就衰。药泉这几年出水越来越弱，支撑两片下品灵田都吃力。田里出货少了，旁支对外换不回多少灵石和药材，宗门那边的供奉却不会因此少半分。

临河坊里，主支陆氏靠宗谱名分压着，临河林氏靠河道和水车卡着，三潮商盟等着压价收货，青岚宗契山院只认供奉和履约，不认你是不是快活不下去。

这摊事不是谁跟谁吵一场就能过去的。

旁支已经被逼到墙角。谁要还想守住这一脉，就得马上站出来扛事。

陆承前抬起头，看向门口那位主支来人，声音不大，却尽量让每个字都落稳。

“并脉以后，我们这一支的人怎么算？”

祖祠里乱糟糟的说话声顿了一下。

那人看了他一眼，像是没想到他刚从草席上爬起来，问的第一句不是求生，也不是叫苦。

“自然归入主册，按族中规矩统管。”

“按什么规矩？”陆承前又问。

那人微微皱眉。

“主支的规矩。”

“我是问，归主册以后，本脉老人、妇孺、无力修行的凡裔，分去哪里，吃什么，用什么，药泉出息归谁，原本在这边侍弄药田的人算旁支旧人，还是主支外役？”

他一口气问完，喉咙发涩，胸口也有些发紧，却没停。

“还有，这祖祠里的牌位，是随你们迁走，还是仍留在此地？若迁走，我们这一支往后祭谁，依谁，香火算在哪一册？”

堂内一下子静了。

刚才还跟着陆正松说话的几个人，此刻都抬头去看门口那人。

这些问题平日里没人会细问，可真问到脸上，谁也不敢当成小事。

门口那位主支来人眯了眯眼，终于正眼看向陆承前。

“少主刚醒，倒是想得很细。”

“想不细不行。”陆承前按着案角站起来，腿还有些虚，可总算站住了，“我这一支要真到了只能投主支活命的地步，也该先知道，投过去以后，剩下的是人，还是账上几行字。”

话落下后，堂里几个原本低着头的人都抬起了眼。

主支来人沉了沉脸色，语气也冷下来几分。

“这些细务，入册之后自然会有安排。旁支如今自顾不暇，先保住人命，才是正经。”

陆承前听完，没急着接话。

这人答来答去，还是不肯把最要紧的事放到明面上。

他正要再开口，旁边忽然传来一声冷哼。

“承前，你刚醒，少说几句大话。”

陆正松皱着眉看他，语气里带着几分长辈压晚辈的意思，“族里都到这一步了，还谈什么分谁、算谁？主支肯接，是情分。你非要把人问烦了，惹出祸来，谁担得起？”

陆承前转头看他。

陆正松说这话时，皱着眉，声音压得不高，听着倒真像是在替全族着急。可他越是急着把祖祠、药泉、账册一并往外推，陆承前心里那点疑心就越压不下去。

原身死在祠里，主支今夜就来收祠。

陆正松先替主支把“并脉保命”的说辞铺了一遍。

这中间但凡没有人提前通气，陆承前把前世那块烂配电箱生吞了都不信。

陆承前没急着顶回去，只淡淡问了一句：“正松叔既然觉得主支安排妥当，那不如先请这位管事把安排说清。”

“你……”

“总不能连吃住、田契、药泉、祭祠归属都没个说法，就叫全族签字按手印吧？”

堂中几个人的脸色立时就变了。

方才大家嘴上说的是议并脉，可“签字按手印”这几个字一摆出来，事情的分量就不一样了。真要落纸，这一支往后还有没有自己的一本账，就不是嘴上争两句能改的了。

门口那人脸色终于不太好看了。

“并脉是族中大事，不是你一个晚辈一句句盘问就能定的。”

“那也不是你们一句‘为你们好’，就能直接收祠收泉的。”

这句话一出，连陆伯衡都抬眼看了他一眼。

陆承前往前走了两步，尽量让自己站得稳一些。

“主支若真是来救人，就该把条件摆明，把话说透，把后路写清。若这些都拿不出来，却只盯着祖祠、药泉和账册”

他顿了顿，盯着门口那人，一字一句道：

“那这不是救人，是收家底。”

堂里一阵压不住的低声议论。

陆正松脸色一沉，像是想发作，却又顾忌什么，终究没敢第一个翻脸。

门口那位主支来人则盯着陆承前看了几息，忽然冷笑了一下。

“倒是小看了你。”

“不过，旁支眼下有什么资格讲价，你心里最好有数。”

陆承前没退，仍站在案前。

“资格是拿来争的，不是拿来等人赏的。”

这句话刚出口，他胸口便猛地一抽，眼前也晃了一瞬。

这副身体到底太弱，强撑到现在，已经快到极限。

他手指在袖里狠狠掐了自己一把，才没当众露出虚相。

祖祠另一头，陆伯衡终于把拐杖往地上一顿。

那一声不算重，却把满堂的杂声全压住了。

“够了。”

他这一开口，陆正松先闭了嘴，连门口那位主支来人也没再立刻说话。

陆伯衡缓缓抬起眼，先看向陆承前，又看向主支来人，声音又老又哑，却还带着几分未散尽的棱。

“并脉不是小事，不能这么一句两句定。”

门口那人皱眉：“伯衡叔，主支的意思已经很明白了。”

“主支的意思，我听见了。”陆伯衡淡淡道，“可这一支还没彻底咽气。承前既然醒了，这话就不该今晚定死。”

他说到这里，握着拐杖的手紧了紧，像是在心里权衡了好一会儿，才最终把话落下来。

“三日。”

“我给这孩子三日。”

“三日之内，若他拿不出办法，拿不出交代，拿不出让族里继续守祠守泉的底气”

陆伯衡缓缓转头，看向陆承前。

“那这一支，就按族议走。”

“并脉也好，交祠也罢，再无人替你拦。”

祖祠里一片死静。

三日不长，却也足够让堂里的人重新算一遍账。

有人想的是真撑不过三日。

也有人在想，若这少主还是撑不起，趁早并到主支，至少还能保住自己这一房不被拖下水。

陆承前把这些眼神一一扫过，心里反倒更定。

这不是能躲过去的事。

今天他不站出来，明天这祖祠就不再是陆氏旁支的祖祠。药泉、灵田、库房、账册，全会顺着并脉的名义被人拿走。到那时，他这个少主，是死是活都没什么分别。

陆正松神色变幻，像是还想说什么，可当着陆伯衡和主支来人的面，终究没再吭声。

主支那人盯了陆承前一会儿，最后只冷冷甩下一句。

“好。三日便三日。”

“三日之后，若旁支还是这副样子，就别再说主支不给活路。”

说完，他转身带人出了祖祠。

门被风一推，雨气一下子灌了进来，吹得灯火轻轻一晃。

堂里那些族人站着没动，却都不约而同地看向了陆承前。

先前是主支站在门口替他们拿主意，如今三日压下来，所有人的目光便都落到了这个刚从草席上坐起来的人身上。

这里头有没有信，陆承前一时也顾不上细分。至少这会儿，没人再急着把祖祠往外推了。

他刚松下那口气，胸口的闷痛便一下子翻了上来，脚下险些一虚。

陆伯衡瞥了他一眼，没当众说什么，只朝侧后偏殿偏了偏头。

“跟我来。”

陆承前没多问，撑着发沉的身子跟了过去。

偏殿比正厅暗得多，只有一盏半残的油灯亮着。外头的争声和雨声都被隔开，屋里一下子静了许多。

陆伯衡站在一张旧供案边，盯着陆承前看了许久，像第一次重新认识这个晚辈。

“你今日说的话，不像从前的你。”

陆承前心里一紧，面上却没露。

“鬼门关前走一趟，总要长点记性。”

陆伯衡没说信，也没说不信，只慢慢点了点头。

“三日不是长命。”

“你若只是嘴上硬，照样救不了这一支。”

陆承前看着他：“那我总得先知道，这一支还能从哪里下手。”

陆伯衡没立刻答，转身去拨了拨供案后头一块歪斜的木牌。

木牌后面，竟露出一道极窄的暗缝。

陆承前刚看清那缝里的东西，指尖便忽然一烫。

像是有一块冰凉许久的铁，在黑暗里毫无征兆地热了一下。

他下意识把手探过去，碰到了一册薄薄的旧卷。

那东西入手冰凉，边角却像藏着一点极细的热意，从掌心一路钻进骨头里。

陆承前心头猛地一跳。

偏殿里灯光昏黄，雨声压在檐外，一切都安静得过了头。

只有他手里那册残旧得几乎看不出原样的薄卷，正隔着掌心，微不可察地发着热。

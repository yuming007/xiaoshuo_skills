# 关系映射

## 核心关系轴

- 本书的人物关系不只分敌我，要分：
  - 血缘关系
  - 利益关系
  - 情报关系
  - 可交易关系
  - 临时同盟关系
- 第一卷写法重点：
  - 亲人不一定可靠
  - 外人不一定不能用
  - 旧恩旧怨会影响资源流向
  - 关系必须能转化成行动，而不只是标签

## 陆承前的核心关系

### 陆伯衡

- 表层关系：老族长与少主
- 实际关系：试探中的托付关系
- 主要矛盾：
  - 陆伯衡不确定陆承前能不能扛事
  - 陆承前不确定陆伯衡还藏着多少旧账
- 关系走向：从互相试探，走向阶段性共担

### 陆正松

- 表层关系：族中长辈
- 实际关系：争权者与潜在内鬼
- 主要矛盾：
  - 陆正松想保自己的那一系
  - 陆承前要查清资源流失和旧账断口
- 关系走向：第一卷中后段必然翻脸

### 陆清禾

- 表层关系：同族晚辈
- 实际关系：主角最早识别出的内政型人才
- 主要矛盾：
  - 陆清禾早知不少问题，但过去无力改变
  - 她需要确定陆承前是不是又一个空喊话的人
- 关系走向：从试探配合走向真正的心腹与副手

### 陆青岩

- 表层关系：族中青壮
- 实际关系：执行线骨干候选
- 主要矛盾：
  - 他服能办事的人，不服光会说话的人
  - 他对陆正松一系既有旧情，也有积怨
- 关系走向：从冲撞，到服本事，再到愿意替主角压场

### 顾行舟

- 表层关系：坊市商人
- 实际关系：可交易、不可尽信的外部桥梁
- 主要矛盾：
  - 顾行舟押的是收益，不是感情
  - 陆承前需要借他的路，但不能让他捏住命脉
- 关系走向：阶段性互利同盟

### 陆昭衡

- 表层关系：同姓同宗的主支来使
- 实际关系：主支派来卡名分、探底细、收主动权的人
- 主要矛盾：
  - 他要按主支意志把旁支压回可控范围
  - 陆承前要借规矩反过来争喘息空间
- 关系走向：礼貌外皮下的硬碰硬

### 何观澜

- 表层关系：转仓社掌柜
- 实际关系：可合作、但完全看账的物流节点
- 主要矛盾：
  - 主角需要路
  - 何观澜需要确定这条脉不会半路塌
- 关系走向：从短单合作到阶段性互信

### 沈归砚

- 表层关系：坊市代书
- 实际关系：证据链外援
- 主要矛盾：
  - 他只认纸面与价钱
  - 主角要借他把旧账做成可用证据
- 关系走向：谨慎合作

## 家族内部结构关系

### 旁支内部三层人

- 第一层：还愿意跟着走的人
  - 代表：陆清禾、陆青岩
  - 特点：嘴上未必服，事上可以用
- 第二层：观望保命的人
  - 代表：多数普通族人
  - 特点：谁能让他们有饭吃、能修炼，就向谁靠
- 第三层：已经把家当作退路和筹码的人
  - 代表：陆正松一系的既得利益者
  - 特点：最怕规矩、最怕清账、最怕按功分利

### 可拉拢 / 可清洗 / 暂不可碰

- 可拉拢：
  - 陆青岩
  - 陆清禾
  - 部分负责药田和巡守的年轻族人
- 可清洗：
  - 长期挪用库物的小管事
  - 吃两头好处的旧账房
  - 借并脉鼓动恐慌的人
- 暂不可碰：
  - 陆伯衡手里压着的旧秘密
  - 主支陆氏在旁支内部的明暗眼线
  - 与黑石岭矿契直接相关的上一代旧人

## 外部关系图

### 陆氏旁支 vs 主支陆氏

- 矛盾性质：名分冲突 + 资源回收 + 旧谱归属
- 主支目标：
  - 以并脉名义回收祖祠、药泉、灵田
  - 防止旁支这一脉重新执谱
  - 掩住某些旧年断脉真相
- 当前状态：表面是家族内部调停，实际是软吞并

### 陆氏旁支 vs 临河林氏

- 矛盾性质：地利冲突 + 河道压制 + 坊市试探
- 林氏目标：
  - 吃掉陆氏失去控制的药材渠道
  - 借河道和水车卡死药田恢复速度
  - 让旁支成为依附者，而非对手
- 当前状态：试探阶段，尚未全面撕破脸

### 陆氏旁支 vs 青岚宗契山院

- 矛盾性质：供奉规则压制
- 契山院目标：
  - 稳定收供
  - 避免地方势力失控
  - 只认产出与秩序，不管家族内部谁生谁死
- 当前状态：冷压，不主动下场，但每一次催契都可能压死旁支

### 陆氏旁支 vs 三潮商盟

- 矛盾性质：交易压价 + 资金锁喉
- 商盟目标：
  - 低价吃资源
  - 用短债换长控
  - 绑定旁支未来出货权
- 当前状态：可合作，但每一步都带钩

### 陆氏旁支 vs 黑石岭巡矿会

- 矛盾性质：旧矿契、旧账、灰色地带分利
- 巡矿会目标：
  - 搞清旧单去向
  - 追索谁该赔钱
  - 寻找新的稳定利益对象
- 当前状态：危险但有交易空间

### 陆氏旁支 vs 河梁转仓社

- 矛盾性质：运输依赖与结算能力考验
- 转仓社目标：
  - 货能到
  - 钱能回
  - 别惹大势力迁怒
- 当前状态：可争取的中立合作方

## 隐线关系

- 陆长岳留下的旧事，可能同时牵涉：
  - 主支陆氏
  - 黑石岭矿契
  - 祖谱缺页
- 顾行舟与三潮商盟并非铁板一块，他自己也有独立账线
- 林氏与主支陆氏未必真正同盟，双方更像是短期借势

## 第一卷关系推进原则

- 所有关系变化都要由事件推动
- 不写无成本信任
- 不写莫名忠诚
- 任何一个角色靠近主角，都必须有：
  - 他自己的利益
  - 他自己的判断
  - 他承担的代价

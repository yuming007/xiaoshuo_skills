# 题材定位

- 书名：《家族修仙：这一脉，由我来立》
- 简短简介：小区保安队长陆承前，雨夜触电意外穿越，再次睁眼，竟到了修仙世界，成了修仙家族的落魄少主。祖祠将失，药泉将断，族人离心，他只能靠一册残缺祖谱，在绝境中撑住这一脉。所有人都觉得这一脉该散了，偏偏他不肯退，这一脉，由我来立。
- 类型：男频仙侠 / 家族修仙 / 势力经营 / 资源争夺
- 一句话卖点：现代保安队长陆承前，穿成与自己同名的没落修仙家族旁支少主，靠祖谱残卷和一身敢打敢闯的狠劲，把快断掉的一脉重新立起来。
- 单章体量：默认三千字左右，单章尽量只承一拍主要矛盾。
- 核心爽点：
  - 热血厮杀
  - 打强敌
  - 外出冒险
  - 闯险地
  - 探险夺宝
  - 抢资源
  - 对外扩张
  - 保祖地
  - 争执谱
  - 立规矩
  - 清内患
  - 收人心
  - 拿资源
  - 补祖谱
  - 争正统
- 文风目标：通俗、顺口、沉稳、少解释、少空转。
- 主角舞台限制：
  - 陆承前不能长期绑在记账、药田、铺面零碎执行和家务细务上。
  - 这类事务默认由副手承担，正文只保留最必要的一笔交代。
  - 主角的主要戏份要放在战斗、冒险、夺资源、探秘和带队破局上。
  - 前期经营压力只是立脉阶段的底盘，不是主角长期声线。
  - 中后期默认逐步把主角重心转到修炼突破、战斗领悟、夺机缘、外出探险和扩张势力上。
  - 不要反复用“算账、清账、钱路、铺面怎么回本”这类念头当主角后期的主要内心驱动。
- 描写要求：
  - 场景优先立环境，再进人物与冲突；每个主场景开头三句内，至少给出两类环境锚点，优先写湿度、气味、光线、器物和地面状态。
  - 场景描写要拆进动作和对话里，不单独停下来堆景；药田、药浴、夜巡这类场面，要让木桶热雾、水痕、药渣、泥地和风声跟着事务一起出现。
  - 动作描写要密，尽量让对白落在动作里推进，动作优先写起势、发力、受力、落点和余波。
  - 重要动作尽量拆成三到五拍；人物情绪复杂时，优先用二次动作和停顿节点代替内心解释。
  - 重要人物和势力出场时，三句内带服装描写；颜色、布料、袖口、绑腿、腰带、靴鞋和整洁度，要服务身份、处境和劳动状态。
  - 群像场景不能只写“众人如何”，要补衣摆泥水、袖口药渍、鬓角汗气、手背热汽之类可见细节。
  - 术法、武技和血脉异象要写清媒介、气机流向、形态和对周围环境的影响。
- 题材底线：不是历史文，不借用东汉、朝堂、郎署、门禁那套冲突模型。

# 题材定位

- 书名：《家族修仙：这一脉，由我来立》
- 简短简介：小区保安队长陆承前，雨夜触电意外穿越，再次睁眼，竟到了修仙世界，成了修仙家族的落魄少主。祖祠将失，药泉将断，族人离心，他只能靠一册残缺祖谱，在绝境中撑住这一脉。所有人都觉得这一脉该散了，偏偏他不肯退，这一脉，由我来立。
- 类型：男频仙侠 / 家族修仙 / 势力经营 / 资源争夺
- 一句话卖点：现代保安队长陆承前，穿成与自己同名的没落修仙家族旁支少主，靠祖谱残卷和现实型经营思维，把快断掉的一脉重新立起来
- 核心爽点：
  - 保祖地
  - 争执谱
  - 立规矩
  - 清内患
  - 收人心
  - 拿资源
  - 补祖谱
  - 争正统
- 文风目标：通俗、顺口、沉稳、少解释、少空转
- 描写要求：
  - 场景优先立环境，再进人物与冲突
  - 动作描写要密，尽量让对白落在动作里推进
  - 重要人物和势力出场时，要带服装描写
  - 服装描写要服务身份、阶层、处境和现场劳动状态
- 题材底线：不是历史文，不借用东汉、朝堂、郎署、门禁那套冲突模型

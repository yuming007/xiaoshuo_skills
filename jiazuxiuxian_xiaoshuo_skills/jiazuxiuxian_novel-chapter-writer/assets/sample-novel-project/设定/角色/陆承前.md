# 陆承前

- 身份：陆家支脉少主
- 前世：27 岁小区保安队长
- 同名设定：前世与原身同名，都叫陆承前
- 穿越初期状态：
  - 刚落地时对仙侠世界几乎一无所知
  - 原身记忆不是整包继承，而是碎片化、延迟式浮现
  - 需要靠称呼、场景、器物和冲突慢慢拼出原身处境
- 当前核心目标：先活下来，先保住这一脉，再把全族底子养回来，想法压住七十万金币血契。
- 当前修为：
  - 明面口径：外人仍会把他当成原来那个炼气镜后期的空架子少主
  - 真实口径：祖谱认血后，散掉的气机被重新拢起，已到炼气镜巅峰
- 当前状态：
  - 对外必须继续藏拙，不能让主家和陆正松一系先知道他修为抬上来了
  - 已经决定先用祖谱盘清可用人手、泉路和外部敌意，先守住这一脉，再谈后面的守地与执谱
  - 已从祖谱得到“清灵养脉浴”完整药方，眼下最要紧的不是卖药，而是先把满田活药化成全族药浴底盘
  - 他心里已经隐约看见“把效果变成活路”的后手，但当前不急着把盘法讲死，先把全族真正练出变化再说
  - 他会和族人一起下池试药，不把风险只压给别人
  - 内部隐藏身世口径：
    - 陆氏白鹿血脉只是他身上的一层
    - 其母并非普通出身，而是带有上古仙族遗脉
    - 他多次显出的龙形意象，根子不在陆氏本脉，而是承于母系血统
- 当前优势：
  - 会排班、分工、盯执行
  - 擅长止损和稳局
  - 看人下菜碟
  - 对灵脉、阵纹、回路类东西上手快
  - 已被祖谱认血，更容易摸到祖地断点和人手错配
  - 最能把乱人手、乱流程、乱场面，重新排成能出结果的活盘
  - 祖谱识人术让他比旁人更敢把全族拆成多条线一起跑，因为他更清楚谁适合守后线、谁适合压场、谁适合冲阵、谁适合封签守密
- 性格补充口径：
  - 平时盘事偏现实，先看资源、规矩和执行条件
  - 但这不等于冷血或缩手
  - 只要是族人当面被打、被辱、被踩脸面的场面，他第一反应必须是先狠狠干回去，再谈后手
  - 他能忍长线亏，不忍眼前这口恶气；尤其看到自家小辈见血时，动作一定比道理更快
- 主角舞台口径：
  - 陆承前是带队往外闯的人，不是长期蹲在家里盯细账、看药田、守铺面的内务总管
  - 记账、药田、后仓、铺面和零碎经营细务，后续默认由副手承担
  - 他的正文核心戏份应优先落在战斗、冒险、争资源、探秘、对外扩张和强敌对撞上
- 外在形象与动作习惯：
  - 常穿洗得发灰的短打或旧劲装，袖口利落，衣摆不长，方便下井、赶路和临时动手
  - 腰间常挂旧钥、短匕或临时塞进袖袋的纸条，不讲究好看，只讲究顺手
  - 看局时常先不急着开口，而是先扫一圈人脸、脚边、器物和出入口
  - 做决定前常有一个很短的停顿，像是在心里把人手、路线和时辰一并排过
  - 真动手时动作干脆，喜欢先站位、再发力，很少空挥架势
- 当前短板：
  - 根基仍虚，巅峰修为是刚被拢起来的，不是彻底养稳的
  - 家族内部信任不足
  - 外部势力都不把他当回事
- 写作约束：
  - 第 1 到第 3 章不能把他写成刚醒就完整识人认事
  - 他前期的判断必须主要来自现场观察、现实经验和碎片记忆
  - 第 4 章起可以更主动地下判断、点人、分事，但定案仍得靠现场痕迹、实物和对质补齐

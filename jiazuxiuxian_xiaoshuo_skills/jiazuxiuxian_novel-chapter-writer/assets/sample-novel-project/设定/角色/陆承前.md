# 陆承前

- 身份：陆氏旁支少主
- 前世：27 岁小区保安队长
- 同名设定：前世与原身同名，都叫陆承前
- 穿越初期状态：
  - 刚落地时对仙侠世界几乎一无所知
  - 原身记忆不是整包继承，而是碎片化、延迟式浮现
  - 需要靠称呼、场景、器物和冲突慢慢拼出原身处境
- 当前核心目标：先活下来，先保住这一脉
- 当前优势：
  - 会排班、分工、盯执行
  - 擅长止损和稳局
  - 看人下菜碟
  - 对灵脉、阵纹、回路类东西上手快
- 当前短板：
  - 修为不高
  - 家族内部信任不足
  - 外部势力都不把他当回事
- 写作约束：
  - 第 1 到第 3 章不能把他写成刚醒就完整识人认事
  - 他前期的判断必须主要来自现场观察、现实经验和碎片记忆

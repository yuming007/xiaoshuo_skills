# 人工二审差异提炼

## 使用时机

- 当前仓库存在 `人工二次审核过后的正文/`
- 续写同一项目前，需要优先学习人工修订后的表达习惯
- 初稿读起来偏“正文腔、解释腔、太整、太滑”时

## 总体观察

对第 1 章到第 7 章的生成稿与人工二审稿对比后，可以确认几条稳定倾向：

- 人工二审整体以压缩为主，总字数明显下降
- 重点不是删剧情，而是删“多余的写法”
- 被稳定删掉的主要是：空开的气氛、重复比喻、动作后的作者点评、过度整齐的判断句、先说满再行动的解释段
- 被稳定保留的主要是：当章事务、核心因果、关键物证、人物身份差、章末承接点

结论：

- 优化方向不是“写得更漂亮”
- 而是“写得更早入事、更少解释、更少空转、更像活人在现场”

## 一、可读性：人工二审最稳定的改法

### 1. 开头更早入事

人工二审常把开头的两三句气氛压掉，尽快落到：

- 谁在场
- 现在在做什么
- 当章第一拍是什么

执行规则：

- 第一段优先给人物和动作
- 天气、冷暖、灯影、风声只能当锚点，不能抢主位
- 如果删掉一段气氛后，读者仍能顺畅读下去，那段多半该删

### 2. 少一层比喻，少一层回声

生成稿容易出现：

- 一个动作
- 一层比喻
- 再补一句情绪回声

人工二审通常会保留其中一层，删掉另外两层。

执行规则：

- 一段里尽量只留一个真正有增益的形象
- 比喻不能只是让句子显得“像小说”
- 能直说就别再拐一次

### 3. 动作说完就停，不补作者点评

人工二审高频删掉的句子类型：

- “这话说得平平，反倒更重”
- “那不是怎样，更像怎样”
- “这一眼才更沉”
- “这才是真正的……”

这些句子的问题不在语法，而在它替读者做了二次理解。

执行规则：

- 动作、对白、站位已经能说明问题，就不要再加“解题句”
- 读者能自己品出来的分量，不要旁白替他品

## 二、连贯性：人工二审更看重“不断线”

### 1. 转场要快，但不能断

人工二审倾向于压缩过场，但并不是随便跳。

保留的通常是：

- 时间点
- 新地点
- 到场后的第一件事

删掉的通常是：

- 为了转场而转场的气氛句
- 到了新地方却暂时没有事务的空段

执行规则：

- 每次转场都要回答：人到了哪里，马上要处理什么
- 不要只写“次日”“数日后”就算承接完成

### 2. 章与章之间要认上一拍余力

人工二审会优先保住上一章留下的：

- 风险
- 关系温度
- 物证
- 未完问题

执行规则：

- 下一章第一拍必须认上一章最后一拍
- 尤其不能把章末压出来的压力，在下一章开头写没了

## 三、逻辑性：压缩可以，证据链不能断

人工二审虽然大量减字，但保留得最稳的是：

- 谁看见了什么
- 这件事为什么可疑
- 它会伤到谁
- 这一步判断是根据什么来的

这意味着：

- 权谋文可以压味道句
- 不能压掉关键推理桥

执行规则：

- 如果删掉一句后，读者就不知道“为什么会怀疑”“为什么会害怕”“为什么要这么做”，那句不能硬删
- 调查、对簿、门禁、封泥、时刻、证言这类场景，允许短，但必须清楚
- 压缩时优先删修辞，不优先删证据

## 四、通人性：人味来自反应，不来自煽情

人工二审不是一味加温情，而是把“人味”从大段感慨，改成更具体的反应。

更有效的人味通常来自：

- 称呼变化
- 回头看一眼
- 递药、递纸、扶人、停手
- 该说的话没说完
- 小人物先顾自己的后路
- 上位者少说、短说、不接话

执行规则：

- 亲情、恐惧、戒备、敬畏，优先落在动作和停顿上
- 不要把人味写成一大段“他心里发沉”“这一刻更重”“这才是真……”的解释
- 配角不能只是功能人，要有身份感、顾忌和生活气

## 五、人工二审稳定压掉的写法

以下写法一旦多了，就容易被人工二审砍：

- 句内连续转折太多
- “像……”“却……”“便……”等虚词过密
- 开头先空写气氛
- 同一意思换三种说法反复讲
- 对白后再补一句旁白解释
- 明明场面已经成立，还补作者的态度句
- 章末概括式收束
- 角色刚听到一句话，立刻接一段很用力的“句子感反应”
- 为了显得有劲，连续切出一排单独成段的名词短句
- 人物已经做出反应，还要旁白替他总结“本质是什么”
- 没交代触发信息，就直接写主角认出人、看懂局、抓到问题本质

## 六、不要机械照搬人工删法

人工二审提供的是偏好，不是“凡长必删”的铁律。

尤其注意两点：

- 有些情感或因果桥，本来就该留
- 如果只学“删字”，不学“留什么”，最后会把章节写空

所以正确吸收方式是：

- 先保事务和因果
- 再压修辞和尾注
- 最后补一到两个最有生活气的细节

## 七、最小执行清单

写完一章后，至少问自己：

1. 开头前三段里，是不是太晚才让人物真正做事
2. 有没有动作或对白说完后，又补一句作者点评
3. 有没有一个意思靠两到三层比喻和感慨反复加重
4. 有没有压缩过头，把证据链或情绪触发点删断
5. 人味是不是落在了具体反应上，而不是落在解释上

只要其中两项答案是“有”，就回去修。

## 八、本轮人工修改新增结论（第001章至第002章）

### 1. 这轮人工稿不是单纯压缩，而是“第1章压表达，第2章补认知”

- 第1章的主要改法仍然是压缩修辞、压掉作者点评、把句子改得更直给
- 第2章则明显加入了一层“主角当场理解这个世界怎么运转”的直给说明
- 这意味着后续续写不能只学“删”，还要学“在该补认知的时候补认知”，尤其是前三章

### 2. 当前人工稿确认的稳定文面习惯

- 标题单独成行写“第X章 章名”，不额外加 Markdown 标题符号
- 旁白更口语，更愿意直接落在“他此刻怎么想、怎么判断、怎么吐槽”
- 允许主角保留现代人口吻，出现短促反问、吐槽和一点黑色幽默
- 人物和局势常先给快速分类结论，再在后文补动作和证据
- 世界说明优先走资源链、势力链、修行链，不绕文学化大弯
- 解释必须服务当前处境，重点回答“这一步为什么不能退”

### 3. 这一版人工稿最该保留的优点

- 主角更像刚穿来的现代人，不像被修仙世界瞬间同化
- 危机感更直接，读者更快明白主角为什么急
- 世界运行逻辑说得更白，资源和名分怎么压人一眼能懂
- 主角识人、分派、看局的现实型脑回路更外显，更符合保安队长底色
- 压抑场面里偶尔有一点短促吐槽，能让人物更活

### 4. 吸收这套风格时的边界

- 要学口语化和即时感，不要学错字、散乱标点和失控玩梗
- 吐槽要短，要准，只能帮场面增压或显人味，不能抢戏
- 世界说明要嵌在主角当前决策里，不能一停就是整段百科
- 人物先下判断可以，但后面必须补足动作、对白或物证，不可长期只靠主角嘴判案

## 九、本轮人工反馈新增执行规则（第007章拆章后）

- 单章默认约三千字，一章只承一拍主要矛盾，不把两次高潮硬挤在同一章里
- 场景开张时必须给一到两个可触可感的环境锚点，优先落天气、湿度、气味、器物和地面状态，让场面先站稳
- 重要人物出场、对峙、验看、操练和谈判时，要补服装与装束细节，拿布料、颜色、束带、泥污、兵刃和磨损去交代身份与处境
- 动作描写优先写起势、发力、受力、落点和余波，不要只写“他很强”“气势很重”这类结论句
- 术法、武技、血脉异象和护体手段，要写调气路径、借力媒介、形态变化和对周围人、田地、水线、器物的即时影响

## 九、本轮人工修改新增结论（第003章）

### 1. 解释段必须先接眼前问题，再落规则

- 不要人物刚问一句，旁白就直接切成“设定讲义”
- 规则解释必须先回答当下这一步为什么急、为什么不能退，再顺带带出制度
- 例如祖祭、验谱这类内容，要先落在“三日后要过哪一关、主支在卡什么”，不能一上来就是概念定义

### 2. 祖谱共鸣不能写成画外音或系统播报

- 不要写成“像有谁告诉他”“一行行字往外蹦”“忽然明白某概念是什么”
- 共鸣优先落在：
  - 掌心发热发麻
  - 气机走到哪一段顺、哪一段涩
  - 身体对不同走向的直接反应
  - 现场器物、节点、伤势和气路给出的触发
- 允许主角事后补一层判断，但必须先有身体感受和现场依据

### 3. 灵根、道基要先写成“他自己摸出来了什么”

- 灵根不要先写定义句，先写：
  - 哪一路气更容易进
  - 往哪儿走能留住
  - 往哪儿一推就散
- 道基不要先写抽象词，先写：
  - 身体明明有底子，却发虚
  - 气能进去，但留不稳
  - 像架子先搭起来了，里面却还是空的
- 也就是说，先有体感，再允许角色后知后觉地把词对上

### 4. 少用 AI 味太重的高压套词

- 少写甚至禁写反复出现的：
  - 压死
  - 踩死
  - 打成名存实亡
  - 硬生生按死
- 真有高压，要改成更具体的说法：
  - 卡死这场小验
  - 定这脉去留
  - 堵后路
  - 让祠、泉、田、库先过不了关
  - 让这支人连转圜的地方都没有

### 5. 陆家内部对立默认写“主家 / 支脉”

- 陆家这条线里，默认口径固定为：
  - “主家” = 原先那一边的主支陆氏
  - “支脉” = 陆承前所在这一脉
- 正文、章纲、追踪、角色卡和势力卡里，只要是在写这两边的对立，默认都用“主家 / 支脉”
- “主支 / 旁支”保留给两种场景：
  - 泛讲世界里的家族制度分类
  - 宗谱、册契、外部正式规则里必须点明旧制名目时
- 内视角日常叙述的替换优先级：
  - 第一优先：这一脉
  - 第二优先：这一支
  - 第三优先：支脉
  - 第四优先：陆家这支
- 需要把对立双方直接点明时，优先写“陆家主家”“陆家支脉”，不要再把“主支 / 旁支”当正文口头禅

### 6. 第001章这次二次人工微调确认的额外写法

- 第一拍可以把“时间 + 地点 + 现场焦点”一口落稳，不必先分成两三句慢慢铺
- 主角前世生活质感可以更直给一点，允许用一两句俗口把人压进现场处境里，比如他烦谁、穷在哪、平时受什么气
- 高压场面里允许局部更口语的即时反应，比如“咋办啊”“老刘头”这类，但只能落在人味和关系感上，不能满章散开
- 人物初醒、发懵、认不全人的阶段，可以先给直给反应，再补识人信息，不必每次都先来一层整齐判断
- 陆家内部称呼已经定口径，后续续写时要把“主家 / 支脉”当作默认叫法，不再来回摇摆

### 7. 第003章这版人工稿额外确认的稳定写法

- 异象场面要先压成“看见什么”，再立刻落回身体反应
- 升级场面按“先摸门、再试气、再出结果”的顺序写，不要一上来先宣布突破
- 祖谱给信息时，只给眼下最有用的半步，不能写成系统播报
- 新帮手出场要立刻绑定“她能做什么”，例如看账、看田、听水线
- 章末优先收在空位、假账、旧钥、锁孔、潮痕这类实证上，让下一章自然开查

### 8. 第一次战斗的写法边界

- 第一场战斗不能只图热闹，必须直接服务当前事务，比如灭口、阻拦查案、争抢证据口
- 先让读者知道双方为什么打，再写怎么打
- 炼气镜战斗优先写：
  - 谁先察觉
  - 谁先稳住气机
  - 谁先破对方气膜
  - 谁更会借地形和省气
- 扮猪吃老虎的爽点要靠“对手误判主角仍是旧底子”，不是靠主角突然无脑秒人
- 首战翻面最好分两拍：
  - 前半拍先让主角吃一点压力
  - 后半拍再亮出更高一层的真实底子反打
- 打完之后必须留下可推进剧情的后果：
  - 对方遗落的口风
  - 更坐实的嫌疑
  - 新的时间压力

### 9. 第013章到第014章人工稿额外确认的稳定写法

- 开盘与卖货场先让主角一句话把路数亮出来，例如“卖药浴包”，然后立刻接定价、客群和赚钱盘法，不先绕一层抽象经营分析
- 坊市群像优先写具体穷客和街面声口：搬货散修、驮运马夫、小田主、门客、外乡人，允许他们说得更土、更急、更夸，不要把人群统一写成整齐的“顾客反应”
- 当章主题可以从街面事务里直接抬一句价值判断，例如“人人有仙修”；但这句话要先有穷修见效、愿意掏钱、现场起哄这些具体支点，再让主角顺势意识到它的分量
- 对手入局要先给隔街观察、买样闻包、打听库存或等主角离场这类前置动作，再落砸店、压场和见血，不能让敌意凭空跳级
- 铺面与生意秩序戏要反复抓器物：木牌、号盒、柜台、药包、木架、账册、钥匙、炭料。不要空写“生意火爆”，要让秩序是看得见、也能被人砸碎的
- 章末要优先收在一个被践踏的具体物上，比如染血号牌、碎木架、踩进泥里的药包，让下一章一开场就能接火

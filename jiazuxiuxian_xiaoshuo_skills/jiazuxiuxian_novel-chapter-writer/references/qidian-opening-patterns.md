# 起点仙侠家族文开局规律

## 用途

- 本文件用于校准小说的商业化阅读节奏
- 重点服务于第 1 章到第 3 章、卷首章、重写开篇、需要增强追读感的章节

## 前三章分工

### 第 1 章

核心任务：

- 立刻把读者拖进场
- 亮出本书最大卖点
- 让主角在危险、异变或强冲突里落地

常见有效元素：

- 穿越落地
- 家族将散
- 祖地将失
- 祖谱异动
- 原身将死或刚死
- 强敌压门

### 第 2 章

核心任务：

- 把身份落稳
- 让读者看清主角周围的人和规矩
- 给主角一个小范围但真实的现实阻力

常见有效元素：

- 家族议事
- 账目、库房、药园、灵田、药泉这些现实盘子亮出来
- 长辈、二叔、堂弟、族妹、附庸、客卿等关系入场

### 第 3 章

核心任务：

- 让主角做出第一轮清晰决断
- 立下近期目标或路径
- 抛出下一个阶段的门槛和代价

常见有效元素：

- 先保哪块地
- 先抓谁
- 先用谁
- 先争什么权
- 先顶哪一波外压

## 单章节奏规则

即使不是前三章，也尽量保持下面这套节奏：

1. 开头一到三段内要有事，不能空转。
2. 每章至少有一个具体事务：族议、查账、守泉、争地、赴坊、谈价、收客卿、试族人、顶供奉、修灵脉。
3. 每章至少有一个新阻力：人、规矩、资源、时间、关系、旧账、境界差。
4. 每章至少有一个兑现点：上一章留下的问题得到部分回答，或读者终于看到主角怎么做。
5. 每章结尾要把人推向下一章，但不要用硬喊口号的方式收尾。

## 对本项目的适配原则

- 开头不是讲修仙世界设定，而是让陆承前先落进家族危机
- 卖点要尽快露出：保安队长思维、祖谱残卷、旁支将散、家族经营
- 前三章新人物数量宜收，不要一上来就堆太多人名和势力名
- 信息应通过场景和动作释放，而不是设定说明

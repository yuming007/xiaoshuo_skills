# 当前剧情状态

## 用途

本文件用于大纲生成，概括截至第9章正文后的事实。

## 当前进度

- 起点：商鞅车裂后魂穿少年贾诩
- 中段：凉州家门潜修四年，确定入仕路线
- 当前：熹平元年入洛为郎中，已因凉州边报、外朝封事与门簿核查被拖入士宦暗斗，并在第一轮对质后暂时脱身

### 第2章

### 第3章

### 第5章

- 完成光禄勋署、杨元、杨赐与南掖门岗位设定

### 第6章

- 凉州急牒请求急递禁中，外朝封事又要求直达禁中候旨，先后在南掖门相撞
- 主角按先后、符验、封泥与时刻照章留痕
- 杨元与杨赐因此调看门簿，主角被正式拉入暗查

### 第7章

- 主角核对底簿与誊本，发现“急牒先到、封事后至”的次序被故意调换
- 卢植站到杨赐一侧，主张先保底簿，不让事情被一句“宫中误事”糊过去
- 当夜原页失踪，主角被顺手推到嫌疑位置上

### 第8章

- 宫里与外朝都来逼主角按各自需要改口
- 主角没有顺着任何一边开口，而是请调换符簿、递签记、交牌时刻与值守交接，借旁簿自证
- 夜里设局守柜，擒住再来改簿的秦姓署吏，对方吐出“这页簿不是替一个人改的”

### 第9章

- 杨赐主持偏厅对质，王甫、段颎、卢植围着一页门簿和凉州边报各自发力
- 主角只就门下所验、所记、所核作证，不替任何一边补刀
- 宫中口谕落下，小黄门与秦姓署吏先行治罪，凉州军费线上的名字被记下待查
- “法不可弃，锋不可露”在这一轮对质后，正式成为主角的行事准则
- 一封不经门禁便先落案头的短札，将更深的线头主动送到主角面前

## 后续大纲的核心承接点

- 主角已被杨赐、杨元正式记住，也已被更深一层人盯上
- 他已经完成从“留痕自保”到“借程序反制”的第一轮升级，下一步要面对主动找上门的试探
- 宫门、郎署、门簿、凉州边报、军饷封事和“谁在改责任链”仍会是后续章节支点
- 马腾线暂时埋下，可择机回收
